# IAM-Labs-Portfolio
This repository documents hand-on IAM labs used to learn and showcase practical skills in indentity systems. The labs focur on IAM fundamentals, cloud identity, hybrid environments, and access governance.
The goal of this portfolio is to demonstrate the following: design and manage identities securely, enforce least-privilege, implement MFA and SSO, support hybrid identity environments, apply IAM governance

All work was completed in lab environments

The following platforms were used: Microsoft Entra ID (Azure AD), Windows Server Active Directory, Okta, VirtualBox

Skills learned:
  -Identity lifecycle management (Joiner/Mover/Leaver)
  -Role-Based Access Control (RBAC)
  -Least-privilege 
  -MFA
  -Conditional access
  -SSO
  -IAM logging and monitoring
  -Access reviews
  -Zero trust

This repository is for educational purposes only. All configurations were perfomed in a lab environment.
